# 00 — Bootstrap Backend

## Objetivo

Criar a fundação técnica do backend antes de atacar módulos de negócio.

## Checklist

1. criar repositório Git
2. criar solution
3. criar projetos:
   - Api
   - Application
   - Domain
   - Infrastructure
   - Contracts
   - SharedKernel
4. criar projetos de teste:
   - Domain.Tests
   - Application.Tests
   - Infrastructure.Tests
   - Api.Tests
5. configurar references
6. configurar EF Core
7. configurar SQL Server
8. configurar Swagger
9. configurar tratamento global de erro
10. configurar health endpoint simples
11. configurar cobertura
12. configurar Sonar
13. criar primeiro teste de smoke
14. garantir build verde

## Resultado esperado
Backend sobe localmente com swagger e testes base passando.
